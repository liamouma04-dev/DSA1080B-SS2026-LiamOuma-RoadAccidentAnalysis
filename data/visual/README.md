## Project Title
Road Accident and Traffic Safety Analysis Using Python

## Dataset Source
**https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents*

## Number of Rows
The sampled dataset contains 48,214 and 49 variables describing accident conditions and characteristics 

## Dataset Description
Some key variables in the dataset include:

| Variable |	Description |
|-----|-----|
| Severity	| Indicates the severity level of the accident |
Start time	|Date and time when the accident occurred|
City	|City where the accident occurred|
State	|State where the accident occurred|
Weather condition	|Weather conditions during the accident|
Temperature	|Temperature at the time of the accident|
Visibility	|Visibility distance during the accident|
Humidity	|Humidity level|
Wind speed	|Wind speed during the accident|
description	|Description of the accident event|

## Research Questions
The following research questions will guide the analysis in this project:
1.	Which locations or states experience the highest number of road accidents?
2.	What time of day has the highest accident frequency?
3.	How do weather conditions influence accident severity?
4.	Which months or seasons have the highest number of accidents?
5.	What is the distribution of accident severity levels in the dataset?

## Expected Outcome
The project is expected to identify meaningful patterns in road accident data, including high-risk locations, peak accident periods, and environmental conditions associated with accidents. These insights may help improve awareness of traffic safety issues and highlight potential areas for safety improvement.

## Key insights
1. Accidents are concentrated in certain states.
2. Peak hours (morning and evening) have the highest accidents.
3. Severity varies depending on time and conditions.
4. Weather conditions influence accident severity.
5. Weekly patterns affect accident frequency.
